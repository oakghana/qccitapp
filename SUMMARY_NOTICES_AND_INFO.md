# IT-APP: Simple Summary of Notices, Info, and Instructions

This summary provides a quick, easy-to-read overview of all important notices, info, and instructions found across modules, forms, and pages in the IT-APP system. All content is simplified and grouped by feature.

---

## 1. Bulk Import (Devices & Repairs)
- Upload CSV files to add many devices or repairs at once.
- Success and error messages appear after each import.
- Download error reports for details if any row fails.
- Only one CSV file can be uploaded at a time.

## 2. Store Management
- Admins, IT Heads, and Store Heads can edit, merge, or delete items.
- Stock Balance Report shows opening, receipt, issues, and closing balances.
- Filter by location and date range.
- Export reports to CSV or PDF.
- Regional IT Heads can now view all regions' stock balances.

## 3. Device Assignment
- Assign devices to staff quickly.
- Search and filter by device or staff.
- See summary cards for unassigned devices and available staff.

## 4. IT Forms & Approvals
- Submit requests for new gadgets, repairs, or stock.
- Approval status is shown for each request.
- Notifications are sent for approvals, rejections, or comments.

## 5. Notifications
- Toast messages show for all important actions (success, error, warning, info).
- Email notifications sent for key events (e.g., import complete, ticket assigned).
- Notification bell shows unread alerts.

## 6. General Info
- Color-coded info for quick scanning.
- All actions are logged for audit trail.
- Real-time updates for critical issues.
