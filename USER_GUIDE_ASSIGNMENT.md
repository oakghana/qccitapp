# How to Use the New User Assignment Feature

## Quick Start Guide

### For Stock Assignment with User Management

1. **Navigate to "Assign Stock to Staff"** in your IT Store module
2. **Click "Assign"** on any stock item
3. In the assignment dialog, you'll see:
   - A list of all users (with Active/Inactive status badges)
   - An "Add New User" button in the top right of the staff selection area

### Adding a New User

1. Click the **"Add New User"** button
2. A dialog will appear with two fields:
   - **Full Name**: Enter the complete name of the person
   - **Email Address**: Enter a valid email address
3. Click **"Create User"**
4. The system will:
   - Create the user in the system as **Active** ✓
   - Automatically populate their details in the assignment form
   - Make them available for immediate assignment

### Viewing User Status

- **Active** (Green badge): User is active and ready for use
- **Inactive** (Gray badge): User is inactive but can still be selected for assignments

## Important Notes

- Only these roles can create new users:
  - Admin
  - IT Store Head
  - Regional IT Head

- New users are automatically created as **Active**
- No email duplicates are allowed
- The email field is required and must be valid
- All user fields are preserved in the system for tracking and reporting

## Example Workflow

```
1. Manager opens "Assign Stock to Staff"
2. Selects HP WIRELESS MOUSE item
3. Clicks "Assign"
4. In the dialog, staff list appears showing all users:
   - John Doe (Active)
   - Jane Smith (Active)
   - Bob Johnson (Inactive)
5. New person arrives: "Need to assign to Sarah Williams"
6. Click "Add New User"
7. Enter:
   - Name: Sarah Williams
   - Email: sarah.williams@company.com
8. Click "Create User"
9. Sarah appears automatically in the staff dropdown
10. Click her name to auto-populate the form
11. Continue with assignment as normal
```
